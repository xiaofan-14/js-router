About JS Router

What is JS Router?

JS Router is a modern, lightweight JavaScript SPA (Single Page Application) router inspired by Vue Router. It provides a declarative way to handle client-side routing with support for dynamic parameters, nested routes, and programmatic navigation.

Usage Example

import { createRouter, defineRoutes } from '@js-router/lib'

const routes = defineRoutes([
    {
        path: '/',
        component: () => '<h1>Home</h1>'
    },
    {
        path: '/user/:id',
        template: '/templates/user.html'
    }
])

const router = createRouter({ routes })
