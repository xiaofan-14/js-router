About JS Router

What is JS Router?

JS Router is a modern, lightweight JavaScript SPA (Single Page Application) router inspired by Vue Router. It provides a declarative way to handle client-side routing with support for dynamic parameters, nested routes, and programmatic navigation.

Key Features

🎯 Dynamic Route Matching

Support for dynamic segments like /user/:id and optional parameters like /posts/:id?

🧩 Web Components

Built-in <route-view> and <route-link> components for declarative routing

⚡ Programmatic Navigation

Navigate programmatically with router.push(), router.replace(), and history methods

🎨 Active Link Styling

Automatic active class management for navigation links

📄 Flexible Content Loading

Support for HTML templates, component functions, and inline HTML content

🔍 Query Parameters

Full support for query string parameters and hash fragments

Usage Example

import { createRouter, defineRoutes } from '@js-router/lib'

const routes = defineRoutes([
    {
        path: '/',
        component: () => '<h1>Home</h1>'
    },
    {
        path: '/user/:id',
        template: '/templates/user.html'
    }
])

const router = createRouter({ routes })
