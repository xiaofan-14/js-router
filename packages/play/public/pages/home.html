Welcome to JS Router Demo

This is a demonstration of the modern JavaScript SPA router with Vue Router-style features.

Features

  • ✅ Dynamic route parameters (e.g., /user/:id)
  • ✅ Web Components (route-view, route-link)
  • ✅ Programmatic navigation
  • ✅ Route guards and meta information
  • ✅ Query parameters support
  • ✅ Active link styling
  • ✅ Template loading from HTML files
  • ✅ Component-based routing
